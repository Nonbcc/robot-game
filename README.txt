ขั้นตอนการ run file robot game (single player)
1.click ไปที่ folder src
2.click ไปที่ folder main
3.click ไปที่ file Main.java
4.กดปุ่ม Run java เพื่อทำการ run file ขึ้นมา

Key binding (ปุ่มในการเล่น)
- กดปุ่ม W A S D เพื่อทำการควบคุมตัวละครไปยังทิศทางต่างๆ
- กดปุ่ม spacebar เพื่อทำการยิงกระสุน

วิธีการเล่น
- เมื่อ run file robot game ขึ้นมาตัวละครจะอยู่ที่ ณ ตำแหน่งที่ X:20 Y:16 ของ map (ขนาด 80x100)
- เป้าหมาย: เดินหลบระเบิด, ยิงระเบิดและสามารถเก็บ energy tank เพื่อเพิ่มพลังชีวิตไม่ให้เหลือ 0 ได้
- เริ่มแรกเราจะมีพลังชีวิตอยู่ที่ 100 หากเหยียบโดนระเบิดพลังชีวิตจะ -5 ต่อระเบิด 1 ลูก
- ถ้าเหยียบระเบิดแล้ว ระเบิดจะหายไปและไปสุ่มเกิดที่ตำแหน่งใหม่
- เมื่อเหยียบระเบิดจนพลังชีวิตเหลือ 0 จะทำให้เกมจบลง
- ในเกมจะมี energy tank ที่สามารถเพิ่มพลังชีวิต +5 ได้ (โดยที่แต่ละ energy tank จะมีความจุไม่เท่ากัน)
- ถ้าความจุใน enerygy tank เหลือ 0 จะหายไป และจะสุ่มสร้างใหม่ที่ตำแหน่งใหม่
- สามารถยิงให้ระเบิดหายไปได้ (เมื่อหายไปจะสุ่มสร้างตำแหน่งใหม่)
- ไม่สามารถยิงให้ enerygy tank หายไปได้ (เมื่อยิงกระสุนจะหายเข้าไป)
- ต้นไม้กับน้ำจะไม่สามารถเดินผ่านได้

ขั้นตอนการออกแบบโปรแกรม
1.เริ่มวิเคราะห์งานจากโจทย์ที่ได้รับหมอบหมาย
2.คิด(รูปแบบ)ทั้งตัวละละคร แผนที่ของเกมที่จะทำ
3.สร้างตัวละคร สร้างแผนที่
4.หาวิธีการทำเกม
5.เริ่มcode
6.เริ่มจากการวาดตัวละครลงไปแล้วสามารถเดินได้อย่างอิสระ
7.ทำการวาด map ลงไป กำหนด scale ที่สามารถเดินได้
8.จากนั้นสร้าง object ที่เป็นระเบิด ทำเรื่องการชนให้ชนระเบิดแล้วจบเกม
9.สร้างเลือดของตัวละคร ถ้าหากเดินชนระเบิดเลือดจะลด
10.สร้าง Energy tank ที่หากไปเดินชนตอนที่เลือดไม่เต็มจะสามารถเพิ่มเลือดได้
11.เริ่มทำกระสุนที่สามารถยิงได้รอบทิศ หากกระสุนลอยไปชน object จะทำให้ object นั้นหายไป
12.พยายามที่จะออกแบบ multiplayer แต่เกิดบัคมากมายจนไม่สามารถแก้ไขได้

Library 
java.io.BufferedReader, java.io.InputStreamReader
ใช้ในการอ่านข้อมูลจากไฟล์ใน java 
java.io.IOException
เป็นฟังก์ชันที่หากข้อผิดพลาดเกิดขึ้นในขณะรันโปรแกรมภาษาจาวาก็จะเรียกใช้งานฟังก์ชันนั้นๆ
java.io.InputStream
เป็นตัวที่จะนำข้อมูลเข้ามาในโปรแกรมที่เราเขียน อาจะจะมาจาก memory ,file หรือ network
java.awt.Color
เป็น package ที่ใช้ในการสร้าง GUI ซึ่งเกี่ยวกับสี
java.awt.Font
เป็น package ที่ใช้ในการสร้าง GUI ซึ่งเกี่ยวกับฟอนท์ตัวหนังสือ
java.awt.Dimension
ใช้สร้าง object โดยเก็บค่า ขนาดความกว้าง และความสูง ไว้ที่ 0, 0
java.awt.event.KeyEvent, java.awt.event.KeyListener
เป็น Interface ในการ ดักฟังเหตุการณ์ ของการทำงานของ keyboard
java.awt.Graphics, java.awt.Graphics2D
เกี่ยวข้องในส่งของกราฟฟิกทั้งหมด เช่น การวาด map ลงไปในตัวเกม การจัด font เขียนตัวษรลงไปบน map ละอื่นๆ
java.awt.image.BufferedImage
ไว้สร้าง BufferedImage ขึ้นมาใหม่สำหรับ Save ภาพให้เป็นไฟล์ใหม่โดยต้องระบุ size และ type
java.awt.Rectangle
ใช้สร้าง object rectangle และเก็บค่าแกน x, y และขนาดความกว้าง ความสูง ไว้ที่ 0 ทั้งหมด
java.util.ArrayList
ใช้เก็บข้อมูลเหมือนอาเรย์ปกติ แต่สิ่งที่แตกต่างกันคือ Array list สามารถย่อและขยายของมันได้อัตโนมัติตามข้อมูลที่มีอยู่ภายใน
java.util.Random
ใช้ในการสุ่มตัวเลขเพิ่มนำมาสุ่มตำแหน่งการเกิดของตัวละครหรือสุ่มจุดเกิด object ต่างๆ
javax.swing.JFrame
เป็น Class หลักที่นิยมนำมาใช้สร้าง Form กรอบที่เป็น Window
javax.swing.JPanel
ไว้สำหรับจัดกลุ่มของ Component Controls ให้อยู่ในกลุ่มเดียวกัน เพื่อใช้ง่ายต่อการแสดงผล และการนำไปใช้งาน
javax.imageio.ImageIO
ใช้ในการเปิดเเละอ่านไฟล์ภาพในรูปแบบต่างๆได้ทั้ง GIF,JPG,PNG
