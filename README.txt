ขั้นตอนการ run file robot game (single player)
1.click ไปที่ folder src
2.click ไปที่ folder main
3.click ไปที่ file Main.java
4.กดปุ่ม Run java เพื่อทำการ run file ขึ้นมา

Key binding (ปุ่มในการเล่น)
- กดปุ่ม W A S D เพื่อทำการควบคุมตัวละครไปยังทิศทางต่างๆ
- กดปุ่ม spacebar เพื่อทำการยิงกระสุน

วิธีการเล่น
- เมื่อ run file robot game ขึ้นมาตัวละครจะอยู่ที่ ณ ตำแหน่งที่ X:20 Y:16 ของ map (ขนาด 80x100)
- เป้าหมาย: เดินหลบระเบิด, ยิงระเบิดและสามารถเก็บ energy tank เพื่อเพิ่มพลังชีวิตไม่ให้เหลือ 0 ได้
- เริ่มแรกเราจะมีพลังชีวิตอยู่ที่ 100 หากเหยียบโดนระเบิดพลังชีวิตจะ -5 ต่อระเบิด 1 ลูก
- ถ้าเหยียบระเบิดแล้ว ระเบิดจะหายไปและไปสุ่มเกิดที่ตำแหน่งใหม่
- เมื่อเหยียบระเบิดจนพลังชีวิตเหลือ 0 จะทำให้เกมจบลง
- ในเกมจะมี energy tank ที่สามารถเพิ่มพลังชีวิต +5 ได้ (โดยที่แต่ละ energy tank จะมีความจุไม่เท่ากัน)
- ถ้าความจุใน enerygy tank เหลือ 0 จะหายไป และจะสุ่มสร้างใหม่ที่ตำแหน่งใหม่
- สามารถยิงให้ระเบิดหายไปได้ (เมื่อหายไปจะสุ่มสร้างตำแหน่งใหม่)
- ไม่สามารถยิงให้ enerygy tank หายไปได้ (เมื่อยิงกระสุนจะหายเข้าไป)
- ต้นไม้กับน้ำจะไม่สามารถเดินผ่านได้
